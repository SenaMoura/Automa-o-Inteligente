# Rastreador-Inteligente
Aprendendo a mexer com Request, smtplib  e base64

Código feito em python com o objetivo de rastrear os preços de e-commerces, identificar quais deles estão acessíveis e ótimos para serem comprados, o usuário terá que seguir os seguintes passos para que o programa funcione de acordo:

(1) Colocar as URLs dos produtos 
(2) Colocar o valor desejado em preco_desejado para que assim consiga achar produtos com um valor de seu agrado
(3) Coloque o seu email para que ele receba mensagens dos produtos que vc deseja comprar com descontos.
(4) Mude a frequência de checagem de preços no While True. 

Antes de começar a fazer esses processos, baixe o zip  aperte com o botão direito do mouse no arquivo, e vai na opção descompactar arquivos.
